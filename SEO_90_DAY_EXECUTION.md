# 90-Day SEO Execution (Renewly)

## Goals
- Primary keyword theme: `contract renewal tracker`
- Audience: US SMB teams
- Scope: technical SEO + homepage on-page SEO

## Weekly Operating Cadence
1. Open Google Search Console > Performance.
2. Filter by homepage URL and keyword cluster queries.
3. Track:
   - impressions
   - clicks
   - average CTR
   - average position
4. Open Indexing reports:
   - Pages
   - Sitemaps
   - Crawl stats
5. Open Core Web Vitals report and PageSpeed Insights for homepage.
6. Log changes and outcomes before shipping the next SEO copy/meta update.

## KPI Targets (Realistic)
- Day 45: clear non-brand impression growth for target cluster.
- Day 60: at least one long-tail keyword reaches top-20.
- Day 90: stable movement toward top-10/top-5 for best-fit long-tail terms.
- CWV mobile p75 targets:
  - LCP < 2.5s
  - INP < 200ms
  - CLS < 0.1

## Change Control
- Update one major SEO variable at a time (title, meta description, or key section copy).
- Keep each test live for at least 14 days before judging impact.
- Record:
  - date shipped
  - change summary
  - affected URLs
  - pre/post metrics
